# Memo Web Application

## Overview
ユーザーがログインし、メモを作成・編集・削除できるWebアプリケーション。

## Features
- ユーザー登録 / ログイン
- メモのCRUD
- RESTful API
- Dockerによる環境構築

## Tech Stack
- Kotlin
- Spring Boot
- PostgreSQL
- JPA (Hibernate)
- Docker
- AWS (予定)

## Architecture
- Controller / Service / Repository 構成
- RDBMSによるデータ永続化

## Security
- SQL Injection対策（ORM使用）
- XSS対策（エスケープ）
